VN100 Class Reference

Detailed Description

Usage

  1. Declare an instance of VN100 with constructor VN100() taking SPIClass as argument. E.g. use VN100 vn100(SPI_2); for MBLC
  2. Call VN100::init() with the chip select pin (PB12 on MBLC)
  3. Call VN100::get() with output arguments to get the filtered angles and angular rates.

Example (on MBLC 0.5)

1 #include <SPI.h>
2 #include <VN100.h>
3 
4 VN100 vn100(SPI_2);
5 
6 void setup() {
7  Serial1.begin(115200);
8  vn100.init(PB12);
9  delay(1000);
10 }
11 
12 void loop() {
13  uint32_t tic = micros();
14  tic = micros();
15  float ypr[3], rates[3]];
16  vn100.get(ypr[0], ypr[1], ypr[2], rates[0], rates[1], rates[2]);
17  tic = micros() - tic;
18 
19  Serial1 << tic << "\t" << ypr[0] << "\t" << ypr[1] << "\t" << ypr[2] << "\t";
20  Serial1 << rates[0] << "\t" << rates[1] << "\t" << rates[2] << "\n";
21 
22  delay(10);
23 }
