Koduino Documentation

The repository associated with this library is at https://github.com/avikde/koduino.

This is a library of code for boards developed using the STM32 F3/F4 series of microcontrollers, chosen for their FPU instructions and friendly built-in bootloaders. The microcontrollers currently targeted are:

  • F373xC (full support)
  • F4xxRG (planned)

The API functions here are in general optimized for speed and concurrency, at the expense of being slightly less fool-proof than Arduino (e.g. see pin configuration, PWM output and input).

API Reference

Core functionality

  • Pin configuration - pinMode, ...
  • Digital I/O - digitalWrite, digitalRead, ...
  • Reading analog signals - analogRead, ...
  • PWM output and input - analogWrite, pulseIn, ...
  • Keeping time - millis, micros ...
  • Running code with precise timing - attachTimerInterrupt, ...
  • External (or pin-change) interrupts - attachInterrupt, ...
  • Printing, streaming and strings - Serial1 <<, ...

Communication

  • Serial / USART - FTDI (USB/Serial), XBee, ...
  • SPI - MPU6000, ...
  • Wire / I2C / SMBus - Nunchuck, MPU6050, ...

Basic libraries

  • Storing data in non-volatile memory (EEPROM)
  • Reading a quadrature encoder
  • Data logging: OpenLog driver

Robotics-oriented libraries

  • Basic and advanced Math (+Eigen)
  • Brushless commutation (field-oriented control)
  • IMU drivers and attitude estimation
  • Motor controller library with multi-motor coordination
